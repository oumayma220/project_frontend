Guide des Variables et Modèles Handlebars

Variables disponibles

Ces variables sont accessibles dans vos modèles Handlebars :

  • employeId : Identifiant de l'employé responsable.
  • nomEmploye : Nom de l'employé responsable.
  • produitId : Identifiant du produit commandé.
  • nomProduit : Nom du produit
  • prixUnitaire : Prix unitaire du produit.
  • quantite : Quantité commandée.
  • total : Prix total de la commande.
  • dateCommande : Date de la commande.
  • adresse : Adresse de livraison de la commande.

Blocs d'itération

  • {{ '{' }}{{ '{' }}#each lignes{{ '}' }}{{ '}' }} : Permet de boucler sur chaque élément du tableau lignes.
  • {{ '{' }}{{ '{' }}/each{{ '}' }}{{ '}' }} : Indique la fin de la boucle #each.

Conseils d'utilisation des Modèles

  • Pour afficher une chaîne de caractères (texte), utilisez la syntaxe : {{ '{' }}{{ '{' }}nomProduit{{ '}' }}{{ '}' }}
  • Pour afficher un nombre ou un identifiant, utilisez la syntaxe : {{ '{' }}{{ '{' }}quantite{{ '}' }}{{ '}' }}
  • Pour parcourir une liste d'éléments, utilisez la structure :
    {{ '{' }}{{ '{' }}#each maListe{{ '}' }}{{ '}' }} ... {{ '{' }}{{ '{' }}/each{{ '}' }}{{ '}' }}
